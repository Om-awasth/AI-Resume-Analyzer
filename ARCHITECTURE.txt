================================================================================
DATA FLOW
================================================================================

1. USER UPLOADS PDF
   └─> Flask receives file
       └─> Saved to /uploads/

2. TEXT EXTRACTION
   └─> PyPDF2 extracts text
       └─> Text cleaning (remove special chars, normalize)
           └─> Extract contact info (email, phone)

3. SKILL DETECTION
   └─> Regex pattern matching
       └─> Check each skill from database
           └─> Build detected skills list

4. TF-IDF ANALYSIS
   └─> Vectorize resume text
       └─> Vectorize job description
           └─> Calculate cosine similarity
               └─> Convert to percentage

5. GAP ANALYSIS
   └─> Compare detected vs required
       └─> Identify matched skills (intersection)
           └─> Identify missing skills (difference)
               └─> Calculate match percentage

6. RESOURCE MAPPING
   └─> For each missing skill
       └─> Lookup learning resources
           └─> Add courses, websites, duration

7. RESULTS DISPLAY
   └─> Render result.html
       └─> Show scores, skills, resources
           └─> Provide recommendations

================================================================================
ML/NLP PIPELINE
================================================================================

STAGE 1: TEXT PREPROCESSING
┌──────────────────────────────────────────────────────────────┐
│ Raw PDF → PyPDF2 → Raw Text → Cleaning → Processed Text     │
│                                                              │
│ Cleaning Steps:                                              │
│ • Remove extra whitespace                                    │
│ • Remove special characters                                  │
│ • Normalize spaces                                           │
│ • Preserve important symbols (#, +, .)                       │
└──────────────────────────────────────────────────────────────┘

STAGE 2: FEATURE EXTRACTION
┌──────────────────────────────────────────────────────────────┐
│ Processed Text → Skill Matching → Detected Skills           │
│                                                              │
│ Method: Keyword Matching with Regex                          │
│ Pattern: \b + skill.lower() + \b                            │
│ Database: 80+ predefined skills                             │
│ Result: List of skills found in resume                      │
└──────────────────────────────────────────────────────────────┘

STAGE 3: TF-IDF VECTORIZATION
┌──────────────────────────────────────────────────────────────┐
│                     TfidfVectorizer                          │
│ Parameters:                                                  │
│ • max_features = 1000                                        │
│ • stop_words = 'english'                                     │
│ • ngram_range = (1, 2)                                       │
│                                                              │
│ Resume Text ────> [v1₁, v1₂, ..., v1₁₀₀₀]                  │
│ Job Description ─> [v2₁, v2₂, ..., v2₁₀₀₀]                  │
└──────────────────────────────────────────────────────────────┘

STAGE 4: SIMILARITY CALCULATION
┌──────────────────────────────────────────────────────────────┐
│              Cosine Similarity Formula                       │
│                                                              │
│         v1 · v2                                              │
│ cos(θ) = ───────────                                         │
│         ||v1|| ||v2||                                        │
│                                                              │
│ Result: Score between 0 and 1                                │
│ Conversion: Score × 100 = Percentage                         │
└──────────────────────────────────────────────────────────────┘

STAGE 5: SKILL MATCHING
┌──────────────────────────────────────────────────────────────┐
│ Set Operations:                                              │
│                                                              │
│ Detected = {Python, SQL, Pandas, NumPy, ...}                │
│ Required = {Python, SQL, Tableau, Power BI, ...}            │
│                                                              │
│ Matched = Detected ∩ Required                                │
│ Missing = Required - Detected                                │
│                                                              │
│ Match % = (|Matched| / |Required|) × 100                    │
└──────────────────────────────────────────────────────────────┘

================================================================================
SCORING SYSTEM
================================================================================

DUAL SCORING APPROACH:

1. SKILL MATCH SCORE
   ┌────────────────────────────────────────┐
   │ Matched Skills / Total Required × 100  │
   │                                        │
   │ Example:                               │
   │ 16 matched / 18 required = 88.89%      │
   │                                        │
   │ Interpretation:                        │
   │ • 90-100%: Excellent                   │
   │ • 75-89%:  Strong                      │
   │ • 50-74%:  Moderate                    │
   │ • 25-49%:  Weak                        │
   │ • 0-24%:   Poor                        │
   └────────────────────────────────────────┘

2. TF-IDF SIMILARITY SCORE
   ┌────────────────────────────────────────┐
   │ Cosine Similarity × 100                 │
   │                                        │
   │ Measures semantic similarity between    │
   │ resume text and job description        │
   │                                        │
   │ Interpretation:                        │
   │ • 70-100%: Strong alignment            │
   │ • 50-69%:  Moderate alignment          │
   │ • 30-49%:  Weak alignment              │
   │ • 0-29%:   Poor alignment              │
   └────────────────────────────────────────┘

COMBINED INTERPRETATION:
┌─────────────────────────────────────────────────────────────┐
│ High Skill Match + High Similarity → Excellent candidate    │
│ High Skill Match + Low Similarity  → Add keywords           │
│ Low Skill Match  + High Similarity → Learn missing skills   │
│ Low Skill Match  + Low Similarity  → Major revision needed  │
└─────────────────────────────────────────────────────────────┘
