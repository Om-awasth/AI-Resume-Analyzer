# Resume Analyzer — Local Setup

## Run in 3 Steps

```bash
cd /Users/omawasthi/Downloads/files/resume_analyzer
npm install
npm start
```

Open: http://localhost:8080

## Optional Environment File

```bash
cp .env.example .env
```

## Notes

- Backend is fully JavaScript (`server.js`).
- Data is stored locally in `data/app.json`.
- Resume upload supports PDF files up to 5MB.
